# Minecraft 1.8.7 Web Server Edition
Bu proje, orijinal 1.8.7 sunucu dosyalarıyla entegre çalışan bir Web-based Minecraft simülasyonudur.

## Özellikler
- **Server Properties Sync:** Sunucu ayarlarıyla senkronize dünya.
- **Hitbox & Physics:** Duvarlardan geçilmeyen gerçekçi fizik.
- **Mobile Optimized:** Tablet ve mobil cihazlarda çalışacak şekilde geliştirildi.
